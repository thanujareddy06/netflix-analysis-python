# 📊 Netflix Movies & TV Shows Analysis

## 🔍 Overview
This project analyzes the Netflix dataset to uncover trends, patterns, and insights related to content distribution and growth. The analysis was performed using Python with data cleaning, exploratory data analysis (EDA), and visualization techniques.

---

## 🛠 Tools & Technologies
- Python
- Pandas
- Matplotlib
- Seaborn

---

## 📊 Key Analysis
- Movies vs TV Shows distribution  
- Content added over the years  
- Top countries producing content  
- Most popular genres  

---

## 📈 Key Insights
- Netflix has more Movies than TV Shows  
- Content increased rapidly after 2015  
- USA produces the highest amount of content  
- Drama and International genres are highly popular  

---

## 🧹 Data Cleaning
- Removed missing values  
- Converted date columns into datetime format  
- Standardized column names  

---

## 📁 Dataset
Netflix Titles Dataset (CSV file)

---

## 🚀 How to Run
1. Clone the repository  
2. Open the notebook in Jupyter or VS Code  
3. Run all cells step by step  

---

## 📌 Conclusion
This project helps in understanding how Netflix content has evolved over time and highlights key factors such as geography and genre preferences.
